Help

1. By default the Generate button will produce a 10 symbols long, mixed string (includes uppercase, lowercase and numerals).
2. The Clear button will erase the already produced string and will return you to the Generate button.
3. By marking the Letters Only checkbox, the script will produce a letters only string (includes uppercase and lowercase).
4. By marking the Numerals Only checkbox, the script will produce a numerals only string (0 - 9).
5. By marking the No Consecutives checkbox, the script will produce a string where no two consecutive characters are the same.
6. By marking the Use Specials checkbox, the script will produce a string which contains special symbols (includes #, $, &, *, +, -, @ and _).
7. The length of the produced string can be manually adjusted by marking the Custom Length checkbox and entering the desired length in the newly appeared textbox.
8. The string may also be preceded by a custom prefix combination. This can be done by marking the Custom Prefix checkbox and entering the custom prefix in the newly appeared textbox. Note that the prefix is included in the desired length.
